Detailed Description

Klasa modelowa reprezentująca pojedynczy posiłek.

Przechowuje wartości liczbowe (makroskładniki) oraz opis tekstowy. Wykorzystuje mechanizm enkapsulacji (prywatne pola i publiczne akcesory).
